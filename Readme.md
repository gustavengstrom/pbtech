# Computations for PBTECH paper

`PBCombos.py`: contains the method `gen_latte` which can be used to run Latte from e.g. a notebook.

`compute_latte_combos.py`: contains code for computing specific boundary and parameter combinations. The results are stored in the results directory.

`boundary_cnt_analysis.py`:contains code for analyzing results and doing further computations. Main methods are `compute_boundary_solved_tables` which computes tables showing the number of boundaries solved and their probabilities and `vrep_analys` which calculates the feasible parameter space from vrep files generated by running JupyMake from the gen_latte() method in `PBCombos.py`. Also `auto_gen_non_me_volumes` was intended to automatically calculate non me volumes from existing but was discontinued.


The results directory active sub directories (22/10-2024):

* combo_latte_all_params: Contains results using all params 
* combo_latte_gt: Contains results where other inputs are set to 0
* combo_latte_gt_minland: Contains results where other inputs, land use efficiency in agriculture and timber are set to 0
